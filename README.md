## 目录结构
##### src:
###### --assets  存放静态资源
###### --components 视图相关组件
###### --network 网络请求
###### --plugins 存放插件(element-ui)
###### --router 路由
###### --store vuex
###### --utils 自定义工具
###### --views 视图

## 安装运行
```shell
$ git clone git@github.com:My-Emperor/vue_FreeToMusic.git
$ npm install
$ npm run serve
```
